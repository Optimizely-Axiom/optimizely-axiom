# InlineInput

Seamless textbox for capturing user input with completely customizable visual styles.

## Documentation

### Usage

The `InlineInput` component requires the `label` prop and is typically paired with another component to provide the visual styles.

<Demo component={Usage} />

### Handling changes

Use the `onValueChange` prop to listen for changes to the value.

<Demo component={ChangeUsage} />

### Controlled

Use the `value` and `defaultValue` props to toggle between controlled and uncontrolled usage. And combine it with `onValueChange` to listen for changes to the state.

<Alert intent="warning" mt="16">
  Remove the text from the following input and press `Enter`:
</Alert>

<Demo component={ControlledUsage} />

### Read-only state

Enable the `disabled` prop to display the read-only state of the input field.

<Demo component={DisabledUsage} />

### Appearance

The inline input field will inherit styles from the parent component.

<Demo component={AppearanceUsage} />

### Multiline

Enable the `multiline` prop to allow multi-line text input.

<Demo component={MultilineUsage} />

### Placeholder

By default we re-use the `label` prop for empty placeholder - but this can be customized using the `placeholder` prop.

<Demo component={PlaceholderUsage} />
