# HoverCard

Display helpful text or previews inside a non-modal dialog when hovering over a link or button.

## Documentation

### Usage

<Demo component={Usage} />

### Controlled

Use the `open` and `defaultOpen` props to toggle between controlled and uncontrolled usage. And combine it with `onOpenChange` to listen for changes to the state.

<Demo component={ControlledUsage} />

### Touch devices

Enable the `keepOpenOnActivation` prop to allow opening the card on clicking the trigger to support mobile devices.

<Demo component={TouchUsage} />

## Trigger

### Customize trigger

By default we use the `Link` component for the dialog trigger which accepts all of the existing link props.

We can also completely change the trigger by using `asChild` and passing our own component.

<Demo component={TriggerUsage} />
