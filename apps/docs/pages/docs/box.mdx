## Composition

`Box` component uses the [`@radix-ui` Composition](https://www.radix-ui.com/primitives/docs/guides/composition) approach and accept an `asChild` prop. Be default it renders a `div` element but when `asChild` is set to `true` it will instead pass all props to its child element.
