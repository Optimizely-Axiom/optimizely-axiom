# Colors

`Box` component supports the `background` and `color` props for setting the background and text colors respectively.

## Usage

<ax-demo name="colors-usage" />

## Dark mode

Semantic color names such as `bg.information`, `bg.success`, etc. support light/dark modes out of the box. The `background` and `color` props also support setting `light` and `dark` mode colors separately.

<ax-demo name="colors-dark-mode-usage" />

Append the `dark` class name to the root `<html>` element to switch to dark mode.

```html /dark/
<html class="dark">
  <!-- ... -->
</html>
```
