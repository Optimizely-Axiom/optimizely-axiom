# Getting Started

<Steps>
### Install

```sh
yarn add @optimizely-axiom/react
```

<Callout>
  Make sure to setup transpiling to handle [CSS Imports](/docs/css-imports/).
</Callout>

### Usage

```jsx
import { Text } from "@optimizely-axiom/react";

export function App() {
  return <Text>Hello World!</Text>;
}
```

</Steps>
