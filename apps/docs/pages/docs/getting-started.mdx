# Getting Started

<Steps>
### Install

```sh npm2yarn
npm install @optiaxiom/react
```

<Callout type="info">
  Make sure to setup transpiling to handle [CSS Imports](/docs/css-imports/).
</Callout>

### Usage

```jsx
import { Text } from "@optiaxiom/react";

export function App() {
  return <Text>Hello World!</Text>;
}
```

</Steps>
