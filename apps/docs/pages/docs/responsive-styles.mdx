# Responsive Styles

`Box` and `Stack` components support responsive styles for all layout related props.

Use the object prop notation where the key is the breakpoint name to apply responsive styles.

## Usage

<ax-demo iframe="/docs/responsive-styles-demo/" name="responsive-usage" />

## Breakpoints

Currently the following breakpoints are supported:
