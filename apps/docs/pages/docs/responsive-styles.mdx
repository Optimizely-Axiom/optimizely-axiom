# Responsive Styles

`Box` and `Stack` components support responsive styles for all layout related props.

<Scale
  hidePreview
  keyLabel="Breakpoint"
  valueLabel="Minimum width"
  values={{ base: "0px", ...tokens.breakpoint }}
/>

## Usage

Use the object prop notation where the key is the breakpoint name:

```tsx
import { Stack } from "@optiaxiom/react";

export function App() {
  return (
    <Stack flexDirection={{ base: "vertical", sm: "horizontal" }}>...</Stack>
  );
}
```

Or use the shorter array prop notation with the elements in order of condition:

```tsx
import { Stack } from "@optiaxiom/react";

export function App() {
  return <Stack flexDirection={["vertical", "horizontal"]}>...</Stack>;
}
```
