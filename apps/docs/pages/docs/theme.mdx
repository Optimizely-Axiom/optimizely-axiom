# Theme

Axiom uses **CSS variables** to declare design tokens.

## Usage

All CSS variables are available under the `axiom-*` prefix.

So given a theme object like this:

```ts
const theme = {
  color: {
    "aqua.50": "#e5f7fb",
    "aqua.200": "#9ae5ef",
  },
  space: {
    0.25: "2px",
    1: "8px",
    sm: "12px",
  },
};
```

The following css custom properties will be available:

```css
:root {
  --axiom-color-aqua-50: "#e5f7fb";
  --axiom-color-aqua-200: "#9ae5ef";
  --axiom-space-0-25: "2px";
  --axiom-space-1: "8px";
  --axiom-space-sm: "12px";
}
```
