# Getting Started

## Installation

<Steps>
### Install package

Install the `@optiaxiom/react` package using your package manager of choice:

```sh npm2yarn
npm install @optiaxiom/react
```

### Setup CSS imports

Make sure to setup transpiling to handle [CSS Imports](/getting-started/css-imports).

### Add global provider

Wrap your application with [AxiomProvider](/components/axiom-provider):

```jsx
import { AxiomProvider } from "@optiaxiom/react";

export function App() {
  return <AxiomProvider>{/* Your app code here */}</AxiomProvider>;
}
```

</Steps>

## Usage

### Basic usage

Import components from `@optiaxiom/react` and use them:

```jsx
import { Text } from "@optiaxiom/react";

export function MyComponent() {
  return <Text>Hello World!</Text>;
}
```
