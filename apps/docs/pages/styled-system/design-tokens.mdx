# Design Tokens

Axiom uses **CSS variables** to declare design tokens.

## Usage

All CSS variables are available under the `--ax-*` prefix.

So given a theme object like this:

```ts
const theme = {
  colors: {
    "aqua.50": "#e5f7fb",
    "aqua.200": "#9ae5ef",
  },
  spacing: {
    "2": "2px",
    "8": "8px",
    sm: "12px",
  },
};
```

The following css custom properties will be available:

```css
:root {
  --ax-colors-aqua-50: "#e5f7fb";
  --ax-colors-aqua-200: "#9ae5ef";
  --ax-spacing-2: "2px";
  --ax-spacing-8: "8px";
  --ax-spacing-sm: "12px";
}
```
