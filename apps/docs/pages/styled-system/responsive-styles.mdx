# Responsive Styles

`Box` and `Flex` components support responsive styles for all layout related props.

<Scale
  hidePreview
  keyLabel="Breakpoint"
  valueLabel="Minimum width"
  values={{ base: "0px", ...tokens.screens }}
/>

## Usage

Use the object prop notation where the key is the breakpoint name:

```tsx
import { Flex } from "@optiaxiom/react";

export function App() {
  return <Flex flexDirection={{ base: "column", sm: "row" }}>...</Flex>;
}
```

Or use the shorter array prop notation with the elements in order of condition:

```tsx
import { Flex } from "@optiaxiom/react";

export function App() {
  return <Flex flexDirection={["column", "row"]}>...</Flex>;
}
```
